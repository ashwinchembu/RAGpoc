Account Security and Privacy

Creating a Secure Account

Password Requirements
Create a strong password with:
- Minimum 12 characters
- Mix of uppercase and lowercase letters
- At least one number
- At least one special character (!@#$%^&*)
- No dictionary words
- No personal information (name, birthday)
- Different from other site passwords

Password Strength Indicator
Our system provides:
- Real-time strength feedback
- Suggestions for improvement
- Common password warnings
- Breach database checking
- Estimated crack time display

Two-Factor Authentication (2FA)

Why Enable 2FA
Additional security layer:
- Protects against password theft
- Required for sensitive operations
- Recommended for all users
- Mandatory for orders over $1,000

2FA Methods Available
Choose your preferred method:
- SMS text message
- Authenticator app (Google, Microsoft, Authy)
- Email verification code
- Biometric (Face ID, Touch ID)
- Hardware security keys (YubiKey)

Setting Up 2FA
Easy 5-step process:
1. Go to Account Settings
2. Click "Security"
3. Select "Enable Two-Factor Authentication"
4. Choose your method
5. Verify with test code

Backup Codes
Recovery options:
- 10 one-time backup codes provided
- Store securely offline
- Each code works once
- Generate new codes anytime
- Required if primary method unavailable

Security Best Practices

Protecting Your Account
Recommendations:
- Never share your password
- Use unique password for our site
- Enable two-factor authentication
- Keep email account secure
- Log out on shared devices
- Clear browser history on public computers
- Review account activity regularly
- Update password every 90 days

Recognizing Phishing
Red flags to watch for:
- Emails asking for password
- Urgent action requests
- Suspicious sender addresses
- Misspelled URLs
- Generic greetings ("Dear Customer")
- Unexpected attachments
- Grammar or spelling errors

Legitimate Communication
How we actually contact you:
- Always from @example.com domain
- Never ask for password via email
- Never request sensitive info via email
- All payment requests through secure checkout
- Include your order number
- Personalized with your name
